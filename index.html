DATA CLEANING

Data Analyst

Through meticulous data cleaning, standardization, and optimization, the project aims to enhance the readability, usability, and analytical potential of the data.

Covid 19

Project made in SQL Server

Data Analyst

Welcome to my Covid-19 Data Exploration Project. In this project, I've used SQL Server to dive deep into Covid-19 data, investigating key aspects of the pandemic's impact, including total cases, deaths, and vaccination rates. I've utilized an array of SQL skills such as joins, common table expressions, temp tables, windows functions, aggregate functions, creating views, and converting data types.

I start by filtering out null continents and ordering the data by location and date. I explore the relationship between total cases and deaths, providing insights into the death percentage in countries like Mexico and the United States. The analysis further examines the total cases relative to population size, revealing the percentage of the population infected with Covid-19 in each country.

I also compare the highest infection and death rates per population across countries and continents. The project offers a global perspective by summing up total cases and deaths worldwide, and for individual countries such as Mexico and the United States.

Finally, I delve into the vaccination progress, showcasing the percentage of the population that has received at least one Covid-19 vaccine. This is achieved using a CTE and temp table to perform calculations on data partitioned by location.

This project is a comprehensive exploration of Covid-19 data, offering critical insights into the pandemic's global, continental, and national impacts. Join me as I unravel the data behind one of the most significant health crises of our era.
